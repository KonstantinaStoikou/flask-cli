=======================
 Flask-CLI v0.1.0
=======================

Flask-CLI v0.1.0 was released on July 31, 2015.

About
-----

Flask-CLI is a partial backport of Flask 1.0 new click integration to v0.10.

Installation
------------

   $ pip install flask-cli
